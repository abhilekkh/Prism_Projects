# Task 1 - Git + Markdown

## 1. What is Git?

- Git is a tool which helps to manage code.

- It keeps record of all changes in your project. If we make mistake, we can go back to old version.

- It is also useful when many people are working on same project. Everyone can work without disturbing others.

- Git works in our computer whereas GitHub is website where we upload our code.

---

## 2. How to Push Code to GitHub

These are steps to upload code to GitHub.

### Step 1: Initialize Git

```bash
git init
````

### Step 2: Connect to GitHub

```bash
git remote add origin https://github.com/yourusername/your-repo-name.git
```

### Step 3: Add files

```bash
git add .
```

### Step 4: Commit changes

```bash
git commit -m "Initial commit"
```

### Step 5: Push code

```bash
git push origin main
```

---

## 3. What is Markdown?

Markdown is a simple way to write text with formatting.

We use symbols like #, *, etc.
It is mainly used in README file.

---

## 4. Basic Markdown Syntax

### Headings

```md

## Heading 2
### Heading 3
```

### List

* Item 1

* Item 3

### Code

